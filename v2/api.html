# 全局 API

# version

  • Type：String
  • 只读
  • 简介：查看插件版本号。
  • 应用：
this.$Focus.version;

# $options

  • Type：Object
  • 只读
  • 简介：可以查看当前插件示例基础配置信息。
  • 应用：
this.$Focus.$options;

# pointer

  • Type：Object
  • default：null
  • 只读
  • 简介：获取与当前活跃层级popsign相对应的焦点对象
  • 应用：
this.$Focus.pointer;

# popSign

  • Type：String
  • default：""
  • 只读
  • 简介：对应当前活跃层级标记名称；与 item 和 group 的 popup 属性名称对应；如果未设置popup属性，默认为""
  • 应用：
this.$Focus.popSign;

# direction

  • Type：String
  • default：""
  • 只读
  • 简介：当前响应按键的名称（全小写）；取值有：left、right、up、down、enter、back 等，更多可参考 keyMaps 中的键名
  • 应用：
this.$Focus.direction;

# getSignList

  • Type：Function
  • 参数：
    • {String|Object} sign 分组标记
    • {String} popup（非必传字段） 弹层标记，不传是位默认层级
  • 简介：根据焦点元素或弹层标记及标签标记获取分组列表
  • 应用：
// 获取默认层级中nav标记列表
this.$Focus.getSignList("nav");
// 获取含有sign标记的焦点对应的列表
this.$Focus.getSignList(pointer);

# openPop

  • Type：Function
  • 参数：
    • {String} popup 层级标记，指定层级标记时，为打开对应弹层；传为空字符串时，则为关闭所有弹层，回到默认层级
    • {Function} fn（非必传字段） 焦点初始化方法，默认执行initFocus(0, 0)
  • 简介：传入相应的层级标记，切换到对应的层级。
  • 应用：
// 'popup'为目标弹层的层级的标记名称
this.$Focus.openPop("popup");

# addPopEvent

  • Type：Function
  • 参数：
    • {String} popup 层级标记名称
    • {Object} events 事件集合。包含keyUp、keyDown、keyLeft、keyRight、keyEnter、keyBack等按键及按键长按事件中涉及的按键操作
  • 简介：在打开的弹层中某些事件需要特殊处理时，可以使用该方法增加相应事件的处理逻辑
  • 应用：
this.$Focus.addPopEvent('popup', {keyUp() {}, keydown() {}, ...})

# closePop

  • Type：Function
  • 简介：根据打开层级的顺序，倒序将层级切换到对应层级上，在关闭弹层或切换默认层级时调用;如果默认层级没有焦点信息，则执行initFocus(0, 0)
  • 应用：
this.$Focus.closePop();

# checkFocus

  • Type：Function

  • 参数：

    • {Object} pointer（非必传字段） 传入指定焦点

  • 简介：根据传入焦点内容，设置当前焦点

  • 输出：pointer/undefined，返回undefined时表示传入焦点对象为当前页面焦点对象/未传入焦点

  • 应用：

// pointer不传是默认在当前活跃层级移动焦点
this.$Focus.checkFocus(pointer);

# moveFocus

  • Type：Function

  • 参数：

    • {String} direction 按传入方向up/down/left/right移动焦点
    • {Object} pointer（非必传字段） 按传入焦点所在层级移动对应层级焦点

  • 简介：根据传入焦点的层级以及方向变量移动对应层级的焦点，默认在当前活跃层级移动

  • 输出：pointer/undefined，返回undefined时表示已移动到边界

  • 应用：

// pointer不传是默认在当前活跃层级移动焦点
this.$Focus.moveFocus("up");

# initFocus

  • Type：Function

  • 参数：

    • {Function/Number/pointer} a 焦点筛选条件方法/基准点 X 方向偏移量/指定焦点
    • {Function/Number} b 焦点初始化后的回调/基准点 Y 方向偏移量(参数a为数字时必传，参数a为函数时非必传)
    • {Function} c（非必传字段） 焦点初始化后的回调

  • 简介：通用焦点初始化方法

  • 应用：

// 根据条件初始化
this.$Focus.initFocus(
  () => Boolean,
  () => {
    // do something
  }
);
// 根据位置信息初始化
this.$Focus.initFocus(0, 0, () => {
  // do something
});
// 直接指定焦点初始化（2.0.8版本支持）
this.$Focus.initFocus(pointer, () => {
  // do something
});

# next

  • Type：Function
  • 参数：
    • {Boolean} flag（非必传字段） 是否在限定区域内移动
    • {Function} fn（非必传字段） 跳转到下一行时执行的操作
  • 简介：
    • 如果焦点未抵达页面右侧边界，则操作等同于页面执行right；
    • 如果焦点已抵达页面右侧边界，则操作等同于先执行下，然后将焦点移动到对应行的第一个焦点
      1. 当flag为真：如果焦点含有group标记，则在group限定区域内执行下一个操作
      2. 当flag为假：在当前页面执行下一个操作
  • 输出：pointer/undefined，返回undefined时表示已移动到边界
  • 应用：
// 不换行
this.$Focus.next();
// 换行
this.$Focus.next(true, () => {
  // do something
});

# prev

  • Type：Function
  • 参数：
    • {Boolean} flag（非必传字段） 是否在限定区域内移动
    • {Function} fn（非必传字段） 跳转到上一行时执行的操作
  • 简介：
    • 如果焦点未抵达页面左侧边界，则操作等同于页面执行left；
    • 如果焦点已抵达页面左侧边界，则操作等同于先执行上，然后将焦点移动到对应行的最后一个焦点
      1. 当flag为真：如果焦点含有group标记，则在group限定区域内执行上一个操作
      2. 当flag为假：在当前页面执行上一个操作
  • 输出：pointer/undefined，返回undefined时表示已移动到边界
  • 应用：
// 不换行
this.$Focus.prev();
// 换行
this.$Focus.prev(true, () => {
  // do something
});

# start

  • Type：Function

  • 参数：

    • {Object/Boolean} item（非必传字段） 含有sign标记的焦点对象/布尔值区别是否仅在区域内移动

  • 简介：

    • 未传参或传参布尔值：
      1. 传参为 true，检测焦点区域标记group是否有效、如果有效则根据left操作移动到相应区域内最左侧的焦点；如果group标记无效，则根据left操作移动焦点到页面最左侧焦点
      2. 未传参、传参为 false 或其他数据类型，则根据left操作移动焦点到页面最左侧焦点
    • 传参焦点对象：根据传入的焦点对象所在层级以及sign标记，切换相应sign标记列表的选中状态到页面布局的第一个

  • 应用：

this.$Focus.start();

# end

  • Type：Function
  • 参数：
    • {Object/Boolean} item（非必传字段） 含有sign标记的焦点对象/布尔值区别是否仅在区域内移动
  • 简介：
    • 未传参或传参布尔值：
      1. 传参为 true，检测焦点区域标记group是否有效、如果有效则根据right操作移动到相应区域内最右侧的焦点；如果group标记无效，则根据right操作移动焦点到页面最右侧焦点
      2. 未传参、传参为 false 或其他数据类型，则根据right操作移动焦点到页面最右侧焦点
    • 传参焦点对象：根据传入的焦点对象所在层级以及sign标记，切换相应sign标记列表的选中状态到页面布局的第一个
  • 应用：
this.$Focus.end();

# clearFocus

  • Type：Function
  • 简介：清除当前层级焦点内容
  • 应用：
this.$Focus.clearFocus();

# checkSign

  • Type：Function

  • 参数：

    • {String} sign 节点标记名称
    • {Function} fn 返回布尔类型来标记当前焦点选中状态
    • {String} popup（非必传字段） 节点所在层级标记

  • 简介：在目标层级上根据Function返回条件切换sign标记的选中状态，默认为当前活跃层级

  • 应用：

// "nav"：类似导航列表添加的标记名称
this.$Focus.checkSign("nav", (data) => {
  return data.id === "1";
});

# getSign

  • Type：Function

  • 参数：

    • {String} sign 节点标记名称
    • {String} popup（非必传字段） 节点所在层级标记

  • 简介：获取当前标记类型选中状态的焦点对象

  • 输出：pointer/null，返回null时表示未设置选中状态

  • 应用：

this.$Focus.getSign("sign");

# nextSign

  • Type：Function

  • 参数：

    • {String} sign 节点标记名称
    • {String} direction（非必传字段） 方向（默认为向右移动）
    • {String} popup（非必传字段） 节点所在层级标记

  • 简介：仅切换目标popup层级sign标记的选中状态，不影响页面焦点

  • 输出：pointer/null，返回null时表示已移动到边界

  • 应用：

this.$Focus.nextSign("sign");

# prevSign

  • Type：Function

  • 参数：

    • {String} sign 节点标记名称
    • {String} direction（非必传字段） 方向（默认为向左移动）
    • {String} popup（非必传字段） 节点所在层级标记

  • 简介：仅切换目标popup层级sign标记的选中状态，不影响页面焦点

  • 输出：pointer/null，返回null时表示已移动到边界

  • 应用：

this.$Focus.prevSign("sign");
