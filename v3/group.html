# group

# 简介

对应焦点注册的 group 标记；用于划分部分焦点所属区域

# 属性

名称 描述 类型 必要性
sign 区域名称标记（对应区域内部节点的group属性，且同一routeview中具有唯一性） String Y
group 所属区域标记（对应父 group 区域的sign属性；默认为''） String N
popup 所属弹层标记 String N

注意

如果group有值，且指向的 group 未被注册；那么对应该group的所有 group 区域在移动计算时将被忽略

# 示例

<template>
	<div
		class="vertical"
		v-group.sign.popup="{ def: outer, sign: outer, popup: popup }"
	>
		<div
			class="vertical-items"
			v-items.group.popup.focus.blur="{
				def: info,
				focus: focus,
				blur: blur,
				popup: popup,
				group: outer,
			}"
		>
			{{ info.msg }}
		</div>
		<div
			class="vertical"
			v-group.sign.group.popup="{
				def: inner,
				sign: inner,
				group: outer,
				popup: popup,
			}"
		>
			<div
				class="vertical-items"
				v-items.group.popup.focus.blur="{
					def: innerInfo,
					group: inner,
					popup: popup,
					focus: focus,
					blur: blur,
				}"
			>
				{{ innerInfo.msg }}
			</div>
		</div>
	</div>
</template>
<script>
export default {
	data() {
		return {
			info: {
				msg: "outer示例文本",
			},
			innerInfo: {
				msg: "inner实例文本",
			},
			outer: "outer",
			inner: "inner",
			popup: "warning",
		};
	},
};
</script>
