# 按键事件

按键事件需要根据不同页面交互需求，重置交互方法；命名视图中如有独立交互逻辑，可单独赋值，默认交互逻辑与按键 code 对应关系如下表。

# resetEvent

  • Type：Function
  • 简介：重置所有按键响应逻辑处理方法(仅重置默认层级的交互逻辑)
  • 应用：
const { proxy } = getCurrentInstance() as ComponentInternalInstance;
proxy?.$Focus.resetEvent();

# addEvent

  • Type：Function
  • 参数：
    • {Object} eventObj 添加交互事件响应定制逻辑
    • {string} popup（非必传字段） 弹层标记，不传视为默认层级
  • 简介：添加按键响应
  • 应用：
const { proxy } = getCurrentInstance() as ComponentInternalInstance;

// 默认层级交互定制逻辑
proxy?.$Focus.addEvent({
  keyEnter: () => {},
  ...
})

// 指定弹层交互定制逻辑
proxy?.$Focus.addEvent({
  keyEnter: () => {},
  ...
}, "popup")
