# 全局配置

# focusName

  • Type: String
  • Default：focushover
  • 简介：节点获焦时添加的类名
  • 应用：
import Vue from "vue";
import $focus from "epg-focus-vue";
Vue.use($focus, { focusName: "focus" });

# selectName

  • Type: String
  • Default：curselect
  • 简介：含有sign标记的节点被选中时添加的类名
  • 应用：
import Vue from "vue";
import $focus from "epg-focus-vue";
Vue.use($focus, { selectName: "select" });

# limitTime

  • Type：Number
  • default：200
  • 简介：按键响应间隔（单位：ms）
  • 应用：
import Vue from "vue";
import $focus from "epg-focus-vue";
Vue.use($focus, { limitTime: 500 });

# useKeyUp

  • Type：Boolean
  • default：false
  • 简介：处理是否支持长按操作，默认未false不支持，配置true时支持长按操作，同时limitTime设置失效
  • 应用：
import Vue from "vue";
import $focus from "epg-focus-vue";
Vue.use($focus, { useKeyUp: true });

# failoverDist

  • Type：Number
  • default：0
  • 简介：焦点计算过程中上下排版间距计算容错值
  • 应用：
import Vue from "vue";
import $focus from "epg-focus-vue";
Vue.use($focus, { failoverDist: 100 });

# keyMaps

  • Type：Object
  • 简介：['left','up','right','down','enter','back','number','foward','rewind','play','pause','stop','delete']键值识别数组中包含的内容，可以根据按键事件中的键值对应关系调整或增加新的键值
  • 应用：
import Vue from "vue";
import $focus from "epg-focus-vue";
Vue.use($focus, {
	keyMaps: {
		up: 56,
		delete: [8],
	},
});

# includeAttr

  • Type：String
  • default：``
  • 简介：VUEx 执行commit的方法名称字符串，该方法对应处理的是还原include列表；在原本正常跳转需要缓存，但是返回时不缓存的页面，需要在返回操作中将自己从 include 列表中排除（详见keepAlive 应用方案）
  • 应用：
import Vue from "vue";
import $focus from "epg-focus-vue";
Vue.use($focus, { includeAttr: "index,list" });

# inactiveTime

  • Type: Number
  • Default：5 * 60 * 1000
  • 简介：无操作持续时间，最低 5 分钟；单位h\m\s对应时\分\秒；未配置该字段或配置时长低于 5 分钟，则inactiveCallback和InterruptInactive不会触发
  • 应用：
import Vue from "vue";
import $focus from "epg-focus-vue";
Vue.use($focus, { inactiveTime: "5m" });

# inactiveCallback

  • Type：Function
  • default：() => {};
  • 简介：无操作持续时间满足inactiveTime配置时长时进入睡眠状态时执行的操作
  • 应用：
import Vue from "vue";
import $focus from "epg-focus-vue";
Vue.use($focus, {
	inactiveCallback() {
		// do something
	},
});

# InterruptInactive

  • Type：Function
  • default：() => {};
  • 简介：无操作持续时间满足inactiveTime时进入睡眠状态，任意按钮中断该状态后要执行的操作
  • 应用：
import Vue from "vue";
import $focus from "epg-focus-vue";
Vue.use($focus, {
	InterruptInactive() {
		// do something
	},
});

# back

  • Type：Function
  • default：() => { history.back(); };
  • 简介：页面统一返回默认执行方法，有相应赋值的情况下优先级：$Focus.keyBack > back > history.back
  • 应用：
import Vue from "vue";
import $focus from "epg-focus-vue";
Vue.use($focus, {
	back: function() {
		history.back();
	},
});
