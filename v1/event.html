# 按键事件

按键事件需要根据不同页面交互需求，重置交互方法。

# setkeyHandle

  • Type：Function
  • 简介：重置所有按键响应逻辑处理方法
  • 应用：
this.$Focus.setkeyHandle();

# keyLeft

  • Type：Function
  • 简介：按键左，默认对应键值：3, 37
  • 应用：
this.$Focus.keyLeft = () => {};

# keyUp
