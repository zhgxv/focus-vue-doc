# 全局配置

# focusName

  • Type: String
  • Default：focushover
  • 简介：节点获焦时添加的类名
  • 应用：
import Vue from "vue";
import $focus from "epg-focus-vue";
Vue.use($focus, { focusName: "focus" });

# selectName

  • Type: String
  • Default：curselect
  • 简介：节点获焦时添加的类名
  • 应用：
import Vue from "vue";
import $focus from "epg-focus-vue";
Vue.use($focus, { focusName: "select" });

# limitTime

  • Type：Number
  • default：0
  • 简介：按键响应间隔（单位：ms）
  • 应用：
import Vue from "vue";
import $focus from "epg-focus-vue";
Vue.use($focus, { limitTime: 500 });

# failoverDist

  • Type：Number
  • default：50
  • 简介：焦点计算过程中上下排版间距计算容错值
  • 应用：
import Vue from "vue";
import $focus from "epg-focus-vue";
Vue.use($focus, { failoverDist: 100 });

# back

  • Type：Function
  • default：() => { history.back(); };
  • 简介：页面统一返回默认执行方法，有相应赋值的情况下优先级：$Focus.keyBack > back > history.back
  • 应用：
import Vue from "vue";
import $focus from "epg-focus-vue";
Vue.use($focus, {
  back: function() {
    history.back();
  },
});
